About Names List Charts

The Names List Charts provide HTML charts of the characters in Unicode, based on NamesList.txt. The characters added in the last release are in yellow.

  1. Click on the block name to get to a code chart.
  2. Click on a code point in the chart to get the names list. Hovering over the code point shows the name.
  3. Click on Help to get this list.
The names are augmented with some additional symbols. For examples, see Latin Extended-B under 01CA (Ǌ) LATIN CAPITAL LETTER NJ.
  • ↑ Points to the uppercase form.
  • ↓ Points to the lowercase form.
  • ↕ Points to the titlecase form (where different from uppercase).
  • ↔︎ Points to the casefold form (where different from lowercase).
  • ⇛ Indicates the full canonical decomposition (NFD) of the character.
  • ⇝ Indicates the full compatibility decomposition (NFKD) of the character.

Certain characters, especially pictographic symbols, may be displayed with a colorful emoji presentation, depending on the browser's default presentation style for those characters.

To properly view these charts, you need to have an updated browser, and Unicode fonts (such as the Noto Fonts) that cover the characters you are interested in.

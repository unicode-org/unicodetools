About Name Charts

The Name Charts provide an index to Unicode names. Each word in each Unicode character name is extracted, and used as an index for the characters.

  • The characters added in the last release are in yellow.
  • To keep the charts from becoming too large, a 'stop-list' of words are omitted. These are:
    • AND, CAPITAL, CHARACTER, COMPATIBILITY, LETTER, SMALL, WITH
    • All script names
    • All words containing a digit
    • All Hangul Syllables
  • Unlike some of the other charts, tooltips to reveal the names are not included (for compactness). However, if you want to know the name of any particular characters:
    • Copy the character from the cell.
    • Go to http://www.ibm.com/software/globalization/icu/demo/transform
    • Paste in under Input 1
    • Select Output 1: Any - Name

To properly view these charts, you need to have an updated browser, and Unicode fonts (such as the Noto Fonts) that cover the characters you are interested in.

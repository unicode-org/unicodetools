This directory contains the final data files 
for Version 15.0.0 of the Unicode Standard.

The "ucd" subdirectory contains the Unicode
Character Database data files.

The "charts" subdirectory contains an archival set of
pdf code charts corresponding exactly to Version 15.0.0.

The "ucdxml" subdirectory contains the XML version of
the Unicode Character Database.

Zipped versions of UCD data files for Version 15.0.0 are
posted in:

https://www.unicode.org/Public/zipped/15.0.0/
